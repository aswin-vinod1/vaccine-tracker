# Vaccine Tracker (Immunization Log)

A simple full-stack application to manage digital immunization records.
The application tracks vaccines taken and automatically calculates the next due dose or booster date.

This project was built as part of a technical assessment.

---

## Features
- Add vaccine records
- Automatic next-dose calculation
- Status tracking (DUE, OVERDUE, COMPLETED)
- Responsive UI
- REST-based backend

---

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- H2 In-Memory Database

### Frontend
- Vue.js 3
- Axios
- HTML, CSS
